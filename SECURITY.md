# Security Policy

## Supported Versions

Aerial is pre-1.0. Security fixes target the latest released version.

## Reporting a Vulnerability

Please do not open a public issue for a security vulnerability.

Use GitHub private vulnerability reporting on `dcdeniz/aerial` if available, or
email the repository owner listed on GitHub with enough detail to reproduce the
issue.

Include:

- affected version or commit
- operating system
- reproduction steps
- impact
- any suggested fix
