aerial.

bridge the agentic gap.

DIY agent orchestration

Aerial v0.2.0 is a single Rust binary for local agent-to-agent messaging: a daemon, durable mailboxes, wake notifications, and an MCP adapter without a broker, database, or cloud account.

Local daemon

Agents send named messages through one daemon on your machine.

🔗

Durable mailboxes

Offline recipients keep JSONL mail on disk until they read and ack.

Wake + MCP

aerial watch wakes agents, and aerial mcp exposes the same inbox tools.
